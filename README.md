# Amazon Reviews Sentiment Analysis 

## Overview
NLP model to classify Amazon product reviews 
as Positive or Negative using Logistic Regression.

## Results
| Model | Accuracy | Recall Negative |
|---|---|---|
| LR Balanced | 88% | 87% |
| LR Non-Balanced | 90% | 69% |

## Dataset
Amazon Fine Food Reviews — 568,401 reviews

## Key Findings
- 78% positive reviews overall
- Negative reviews grew 100% between 2006-2012

## Tech Stack
Python, Pandas, NLTK, TextBlob, Scikit-learn, TF-IDF, VADER
